Dans le cadre d'un tutoriel Openclassroom j'ai créée un dossier 'test' pour mettre en pratique les notions de base.
En pré-requis le logiciel Mamp (qui contient Apache, MySql et PHP) a été installé.

Dans ce dossier j'ai fais quelques exercices et j'ai noté quelques petits commentaires sur les bases de PHP.

Le fichier Bases.php m'a permis de tester et d'appréhender en PHP les variables, les tableaux, les conditions, les boucles et les fonctions.

Les fichiers entete.php , menu.php et footer.php sont relié à index.php par la fonction include().

Le fichier test.php constitu le premier contact avec le langage PHP dans le cadre du tutoriel Openclassroom.
